Uso en HTML

<!-- Filled -->
<span class="material-icons">school</span>

<!-- Outlined -->
<span class="material-icons-outlined">school</span>

<!-- Con tamaño explícito (tokens del SVG: 18/24/36/48px) -->
<span class="material-icons size-24" style="font-size:24px">person</span>

<!-- Con color SLEP -->
<span class="material-icons icon-royal">info</span>

Alternativa — estilo.md
Lucide Icons (recomendado en estilo.md para componentes)
El SVG de referencia usa Material Icons; estilo.md recomienda Lucide para componentes HTML. Ambos son válidos.

Carga

<script src="https://unpkg.com/lucide@latest"></script>
<script>lucide.createIcons();</script>

Uso

<i data-lucide="graduation-cap"></i>
<i data-lucide="arrow-right"></i>
<i data-lucide="check"></i>
